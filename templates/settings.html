Clear All System Data

Permanently deletes ALL records: students, grades, attendance, fees, announcements, and assignments. User accounts are preserved. This cannot be undone.

Delete Non-Admin Users

Removes all Teachers, Parents, Accountants, and student-linked users. Admin accounts are kept. All related records (grades, attendance) are also removed.

Delete Inactive Users

Removes all user accounts marked as Inactive or Suspended from all role tables. Active accounts are not affected.

Reset System to Default

Wipes ALL settings, user data, and records. Restores factory defaults. System will reload. Equivalent to a fresh installation.

Export All Data

Downloads a complete JSON dump of all settings and data from localStorage. Use this before performing any destructive operation below.

Debug Information

Logs browser details, screen resolution, localStorage contents, and system settings to the browser console for troubleshooting.

Purge Entire Database

This will send a DELETE command to the backend that drops and recreates all tables. Every student, teacher, fee record, grade, and attendance record is permanently gone. Type PURGE EMALASHIRA to enable this button.
